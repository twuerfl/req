## Description

The system shall provide a CodeScanner that scans source files for inline requirement tags using regex patterns.

## Rationale

Automated scanning eliminates manual traceability tracking. Regex-based parsing supports multiple languages and tag formats efficiently.

## Acceptance Criteria

- [ ] Scanner detects REQ: and VERIFIES: tags
- [ ] Supports multiple languages (C, Rust, Python, etc.)
- [ ] Reports file path and line number for each tag
