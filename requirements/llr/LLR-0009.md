## Description

The system shall provide a ReqIF adapter for importing and exporting requirements to/from industry-standard tools like DOORS, Polarion, and Integrity.

## Rationale

ReqIF (Requirements Interchange Format) is the OMG standard for requirements exchange. This adapter enables the tool to integrate with existing enterprise toolchains without vendor lock-in.

## Acceptance Criteria

- [ ] Import requirements from .reqif and .reqifz files
- [ ] Export requirements to ReqIF format
- [ ] Configurable attribute mapping via mapping config
- [ ] Preserve external UUIDs for round-trip support
