## Description

The system shall provide an optional pre-commit hook that warns about traceability issues without blocking commits on development branches.

## Rationale

Pre-commit hooks provide early feedback to developers without creating friction. Advisory mode allows rapid iteration while still raising awareness of traceability gaps.

## Acceptance Criteria

- [ ] `req hooks install` command to install hook
- [ ] `req hooks uninstall` command to remove hook
- [ ] Hook runs `req check` (advisory, non-blocking)
- [ ] Branch-aware: can skip checks on configured branches
