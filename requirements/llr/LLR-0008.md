## Description

The system shall provide configuration management via TOML files for project settings, source directories, and adapter profiles.

## Rationale

Configurable settings enable teams to customize the tool for their project structure and workflow without modifying code.

## Acceptance Criteria

- [ ] TOML-based configuration file (.req/config.toml)
- [ ] Configurable source directories and requirements path
- [ ] Adapter profiles for import/export customization
