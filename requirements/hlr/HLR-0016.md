## Description

The tool shall produce deterministic results for identical repository states.

## Rationale

Determinism is required for CI reproducibility, AI reliability, and auditability.

## Acceptance Criteria

- [ ] Identical input produces identical output
- [ ] No hidden or time-dependent state
- [ ] Results independent of execution environment
