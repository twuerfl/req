## Description

The tool shall construct a complete traceability graph from HLR to LLR to code and tests.

## Rationale

Graph-based traceability enables systematic detection of missing links and inconsistencies required for certification and AI analysis.

## Acceptance Criteria

- [ ] Graph can be visualized in the CLI
- [ ] Graph is deterministic for identical inputs
- [ ] Missing links are reported as gaps
- [ ] Graph export as JSON is supported
