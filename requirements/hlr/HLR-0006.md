## Description

The tool shall export requirement data in structured formats for CI, audit, and external use.

## Rationale

Machine-readable exports enable automation, while human-readable formats support audits and reviews.

## Acceptance Criteria

- [ ] JSON export includes full metadata
- [ ] Markdown export is supported
- [ ] Schema versioning is enforced
