## Description

The tool shall support multiple programming languages via configurable parsing.

## Rationale

Modern projects are polyglot and require flexible tooling.

## Acceptance Criteria

- [ ] Language-independent tagging
- [ ] Configurable parsing rules
- [ ] Language metadata stored per reference
