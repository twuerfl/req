## Description

The tool shall expose its functionality via a machine-accessible interface for AI systems.

## Rationale

AI-assisted development requires direct programmatic access to requirement data and analysis tools.

## Acceptance Criteria

- [ ] Structured API or protocol is available
- [ ] Operations for scan, coverage, gaps, and export are exposed
- [ ] JSON-compatible I/O
