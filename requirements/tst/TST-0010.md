## Description

Test suite verifying traceability graph report generation and impact analysis, covering the full HLR→LLR→Code chain traversal.

## Test Cases

### TC-001: Trace Tree for HLR

Verify forward trace from a root HLR:

- Returns all child LLRs
- Returns all code references for each LLR
- Correctly marks nodes with missing children as gaps

### TC-002: Trace Tree for LLR

Verify trace for a single LLR:

- Returns parent HLR reference
- Returns all code references
- Returns all test references

### TC-003: Impact Analysis — Direct

Verify first-order impact for a changed HLR:

- Returns all directly derived LLRs
- Returns all code files implementing those LLRs

### TC-004: Impact Analysis — Transitive

Verify transitive impact across the full chain:

- HLR change propagates through LLRs to code refs
- No duplicate entries in result
- Result is JSON-serialisable

### TC-005: JSON Report Output

Verify exported report format:

- Output matches `AiExport` schema
- `schema_version` field is present and correct
