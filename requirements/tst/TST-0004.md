## Description

Test suite verifying the SQLite cache layer for requirement storage.

## Test Cases

### TC-001: Cache Initialization
Verify cache database creation:
- Creates .req/cache.db file
- Creates required tables
- Initializes schema

### TC-002: Requirement CRUD Operations
Verify Create, Read, Update, Delete operations:
- Insert requirement
- Query by ID
- Update existing requirement
- Delete requirement

### TC-003: Code Reference Storage
Verify code reference storage:
- Store file path, line numbers
- Link to requirement ID
- Query by requirement ID

### TC-004: File Hash Caching
Verify file hash caching:
- Calculate and store file hashes
- Detect file changes
- Skip unchanged files on re-scan

### TC-005: Link Operations

Verify traceability link storage:

- Store and retrieve HLR→LLR and LLR→Code links
- Query all links for a given requirement ID
