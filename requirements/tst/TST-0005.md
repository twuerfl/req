## Description

Test suite verifying the `TraceGraph` gap detection and coverage metric
calculations. Coverage report formatting (text/JSON output) is tested separately
by TST-0023 (parent: LLR-0018).

## Test Cases

### TC-001: HLR Coverage Calculation
Verify HLR coverage metrics:
- Count total HLRs
- Count HLRs with LLR children
- Calculate percentage

### TC-002: LLR Implementation Coverage
Verify LLR implementation metrics:
- Count total LLRs
- Count LLRs with code references
- Calculate percentage

### TC-003: Orphan Code Detection
Verify orphan code ref detection:

- Code refs pointing to undefined requirement IDs are flagged
- Report count of orphan references

### TC-004: Coverage Report Output
Verify report formatting:
- Text output for CLI
- JSON output for CI integration
