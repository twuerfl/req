## Description

Tests that `req_mcp` meets its architecture constraints: binary named
`req-mcp-server`, rmcp dependency present with correct features.

## Test Cases

### TC-001 — req_mcp/Cargo.toml declares [[bin]] name = "req-mcp-server"
Read `req_mcp/Cargo.toml` and assert it contains `name = "req-mcp-server"` under `[[bin]]`.

### TC-002 — req_mcp/Cargo.toml lists rmcp with server and macros features
Read `req_mcp/Cargo.toml` and assert the `rmcp` dependency includes
`features = [...]` containing `"server"` and `"macros"`.

### TC-003 — cargo build -p req_mcp succeeds
Assert: the build produces no errors. (Verified by CI / `cargo build --workspace`.)

## Acceptance Criteria

- [ ] TC-001 through TC-003 pass under `cargo test`
- [ ] Tests located in `tests/arch_tests.rs`
